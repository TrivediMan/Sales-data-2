# 📊 Sales Performance Dashboard (Power BI)

## 📌 Project Overview
This project presents an interactive **Sales Performance Dashboard** built using Power BI. It provides insights into sales trends, customer behavior, product performance, and regional distribution.

---

## 🎯 Objective
- Analyze overall sales performance  
- Track customer activity  
- Identify top-selling products  
- Compare region-wise sales  
- Understand payment behavior  

---

## 📂 Dataset Details

| Column Name    | Description |
|----------------|------------|
| TransactionID | Unique transaction ID |
| Date          | Date of transaction |
| ProductName   | Product sold |
| Category      | Product category |
| Region        | East, West, North, South |
| SalesChannel  | Online / Offline |
| Quantity      | Units sold |
| UnitPrice     | Price per unit |
| TotalAmount   | Quantity × UnitPrice |
| PaymentMode   | Credit Card, Cash, UPI, Net Banking |
| CustomerID    | Unique customer ID |
